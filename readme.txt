== Description ==
Grouper is a plugin of re grouping of plugins specially developed for the @webstartup plugin.
Once installed, the compatible plugins will automatically be gathered into a single link.
If you do not have a Grouper-compliant plugin, do not look for anything, nothing new will be installed on your back office.

== Installation ==
1. Upload Grouper to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Screenshots ==
1. Activate the plugin
2. Compatible plugins will be linked in a single Grouper menu.
